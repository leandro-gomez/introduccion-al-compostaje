# introduccion-al-compostaje

Introducción al compostaje mediante el juego

## Inicializaión

```bash
npm install
```

### Desarrollo

```bash
# Servir los archivos compilados y recargar si hay cambios
npm run serve

# Verificar estilos
npm run lint
```

### Compilar y minificar para producción

```bash
npm run build
```
